# choke
choke is a small interpreted programming language in C#, built as a practice project to explore language design and runtime concepts.

# How to use?
1. Download the latest release from the [releases page](https://github.com/isaiahnoelsalazar/choke/releases).
2. Extract the downloaded archive.
3. Open a terminal and navigate to the extracted directory.
4. Run the interpreter:
	```bash
	choke.exe Hello world!
	```
